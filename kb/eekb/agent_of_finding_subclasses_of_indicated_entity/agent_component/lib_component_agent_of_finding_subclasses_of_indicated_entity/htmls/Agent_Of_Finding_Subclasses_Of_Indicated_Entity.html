Задачей агента поиска подклассов указываемой сущности является поиск подклассов указываемой сущности. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу поиска подклассов указываемой сущности. Единственным входным аргументом является сущность, для которой требуется найти подклассы.
