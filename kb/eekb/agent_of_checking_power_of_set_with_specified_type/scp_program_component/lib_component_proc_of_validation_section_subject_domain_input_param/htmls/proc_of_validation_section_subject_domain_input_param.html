Задачей scp-программы проверки входных параметров sc-агента на соответствия разделу, элементов, на которые он разбивается, предметной области является проверки входных параметров sc-агента на соответствия разделу, элементов, на которые он разбивается, предметной области. Первым входным параметром данной scp-программы является узел вопроса агента. Вторым входным параметром данной scp-программы является входной параметр агента.
