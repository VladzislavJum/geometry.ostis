Задачей агента поиска обобщённых спецификаций для указываемой сущности является поиск обобщённых спецификаций для указываемой сущности. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу поиска обобщённых спецификаций для указываемой сущности. Единственным входным аргументом является сущность, для которой требуется найти обобщённые спецификации.
